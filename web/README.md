## 📃 Guide 📖

- Install and config Tailwind CSS:

  - npm install -D tailwindcss postcss autoprefixer
  - npx tailwindcss init -p
  - tailwind.config.cjs: content: ["./src/**/*.{html,tsx,css}"],
  - src/styles/global.css:

        @tailwind base;
        @tailwind components;
        @tailwind utilities;
